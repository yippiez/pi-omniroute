# OmniRouter

## Usage

```bash
omnirouter "explain monads"                 # run a prompt
omnirouter -m auto/coding "refactor this"   # pick a model
omnirouter --json "3 primary colors"        # structured JSON output
omnirouter serve --port 8080                # local OpenAI-compatible server
omnirouter --list                           # list models
omnirouter --help                           # all options
```
